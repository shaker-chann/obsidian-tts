# Simple TTS - Obsidian 朗读插件

一个基于 Web Speech API 的 Obsidian 文字朗读插件，支持中文语音朗读。

## 功能特性

- ✨ **朗读功能**：朗读当前笔记内容或选中的文本
- ⏯️ **播放控制**：支持播放、暂停、继续和停止操作
- 🔊 **语音设置**：可调节语速、音调、音量
- 🗣️ **语音选择**：自动选择中文语音或手动选择其他语音
- 📊 **状态栏按钮**：快速访问播放控制
- ⌨️ **快捷键支持**：通过命令面板执行操作

## 安装方法

### 手动安装

1. 下载 `obsidian-tts-plugin/` 文件夹
2. 在 Obsidian 中打开设置 → 第三方插件 → 打开插件文件夹
3. 将插件文件夹复制到插件目录中
4. 在插件设置中启用 Simple TTS

## 使用说明

### 基本操作

1. **播放朗读**：点击状态栏的 ▶ 按钮，或使用命令面板执行「播放/继续朗读」
2. **暂停朗读**：点击状态栏的 ⏸ 按钮，或使用命令面板执行「暂停朗读」
3. **停止朗读**：点击状态栏的 ⏹ 按钮，或使用命令面板执行「停止朗读」

### 设置选项

- **语速**：设置朗读速度（0.5 - 2.0）
- **音调**：设置朗读音调（0.5 - 2.0）
- **音量**：设置朗读音量（0 - 1.0）
- **语音**：选择要使用的语音（自动选择中文或手动选择）

### 快捷键

打开 Obsidian 设置 → 快捷键，搜索以下命令并设置快捷键：

- `simple-tts-play`: 播放/继续朗读
- `simple-tts-pause`: 暂停朗读
- `simple-tts-stop`: 停止朗读

## 技术实现

### 核心技术

- **Web Speech API**：浏览器原生的文字转语音接口
- **SpeechSynthesisUtterance**：语音合成核心对象
- **SpeechSynthesis**：语音合成控制器

### 项目结构

```
obsidian-tts-plugin/
├── main.js          # 插件主代码
├── manifest.json    # 插件配置
└── styles.css       # 样式文件
```

### 代码结构

- `SimpleTTSPlugin`：插件主类，管理朗读状态和控制
- `SimpleTTSSettingTab`：设置页面，管理用户配置
- 核心方法：
  - `getActiveNoteText()`：获取当前笔记文本
  - `startSpeech()`：开始朗读
  - `togglePlayPause()`：播放/暂停切换
  - `pause()`：暂停朗读
  - `stop()`：停止朗读

## 浏览器兼容性

本插件使用浏览器原生的 Web Speech API，需要浏览器支持该特性：

- ✅ Chrome/Edge 支持
- ✅ Firefox 支持
- ✅ Safari 支持
- ✅ Opera 支持

## 注意事项

1. 使用前请确保浏览器支持 Web Speech API
2. 首次使用可能需要等待语音列表加载
3. 建议在网络良好的环境下使用
